# 艾宾浩斯复习计划表 - Apple Store 极简风格设计

## 设计理念

**设计运动**: Apple Human Interface Guidelines - 极简主义与功能美学的完美结合

**核心原则**:
- **清晰性(Clarity)**: 每个元素都有明确的目的，文字清晰易读，图标精确且信息丰富
- **遵从性(Deference)**: 界面服务于内容，流畅的动画和精致的视觉效果帮助用户理解交互
- **深度(Depth)**: 通过分层和动画创造视觉深度，帮助用户理解层级关系
- **极简主义**: 大量留白、简洁的布局、只保留必要元素

**色彩哲学**:
- 主背景: 纯净的白色(#ffffff)和极浅灰(#f5f5f7)
- 强调色: 苹果蓝(#007AFF) - 用于主要操作按钮和链接
- 次要色: 柔和的灰色系(#86868b, #d2d2d7)用于次要文本和边框
- 成功色: 翠绿(#34C759)表示完成状态
- 渐变: 柔和的线性渐变，从浅蓝到浅紫

**布局范式**:
- 中心对齐的单列布局，最大宽度约1200px
- 大量垂直和水平留白(padding: 80px+)
- 卡片式设计，带有轻微阴影和圆角(border-radius: 18px)
- 响应式网格系统，移动端自适应为单列

**标志性元素**:
- 毛玻璃效果(backdrop-filter: blur)用于浮动元素
- 柔和的阴影(box-shadow: 0 4px 20px rgba(0,0,0,0.08))
- 大号标题配合轻量级正文
- SF Symbols风格的图标系统

**交互哲学**:
- 所有交互都应该感觉自然和直观
- 按钮按下时有轻微的缩放效果(scale: 0.98)
- 悬停时元素轻微上浮(translateY: -2px)
- 页面切换使用淡入淡出，持续时间300ms

**动画**:
- 使用ease-in-out缓动函数
- 过渡时间: 快速交互200ms，页面切换300ms，复杂动画400ms
- 元素入场使用淡入+轻微Y轴位移
- 避免过度动画，保持克制和优雅

**字体系统**:
- 标题: -apple-system, BlinkMacSystemFont, "SF Pro Display" (600-700 weight)
- 正文: -apple-system, BlinkMacSystemFont, "SF Pro Text" (400-500 weight)
- 数据: SF Mono或系统等宽字体
- 层级: H1(48px) → H2(36px) → H3(24px) → Body(17px) → Caption(13px)

**视觉资产**:
- 使用柔和的渐变背景
- 简洁的线条图标
- 高质量的产品插图，风格简约现代
- 避免过于复杂的图形，保持视觉清爽
